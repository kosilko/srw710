Дампы микросхемы serial flash MK25D16ATIG с китайской и английской версий радиоприемника.
100% взаимозаменяемы.
Сняты с помощью USB программатора CH341A. Им-же китайская версия прошивки была успешно заменена на английскую.
У английской версии навигация по меню работает наоборот (кнопка влево = вниз, кнопка вправо = вверх). Баг касается именно навигации по меню, перемотка вперед/назад работает нормально.
